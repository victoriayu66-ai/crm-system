# 客户管理系统 - 使用指南

## 🎯 快速开始

### 1. 打开系统
直接在浏览器中打开 `index.html` 文件即可使用。

### 2. 登录系统

#### 使用默认管理员账号（推荐）
- **邮箱**：admin@fengfancloud.com
- **密码**：admin123
- **角色**：管理员（可查看所有数据）

#### 注册新账号
1. 点击"立即注册"
2. 填写姓名、企业邮箱（必须是 @fengfancloud.com）、密码
3. 新账号默认为**普通用户**角色
4. ⚠️ **重要**：仅支持公司企业邮箱（@fengfancloud.com）注册，其他邮箱将被拒绝

---

## 👥 角色权限说明

### 管理员角色
✅ 可以查看所有用户创建的客户  
✅ 可以添加客户，无查重限制  
✅ 可以查看全局统计数据  
✅ 可以管理所有商机和交互记录  

### 普通用户角色
🔒 只能查看自己创建的客户  
🔒 添加客户时自动检查邮箱和电话是否重复  
🔒 只能看到自己客户的统计数据  
🔒 只能管理自己的商机和交互记录  

---

## 📋 功能演示

### 场景1：管理员查看所有数据
1. 使用 admin@fengfancloud.com 登录
2. 进入"客户列表"，可以看到系统中所有客户
3. 进入"数据概览"，可以看到全局统计数据
4. 添加客户时无需查重

### 场景2：普通用户添加客户（带查重）
1. 注册新账号（例如：zhangsan@fengfancloud.com）
2. 登录后进入"客户列表"
3. 点击"添加客户"
4. 填写客户信息：
   - 姓名：测试客户
   - 邮箱：test@example.com
   - 电话：13800000001
5. 点击"保存"，客户添加成功
6. 再次尝试添加相同邮箱或电话的客户
7. 系统提示"该客户已存在（邮箱或电话重复），无法添加！"

### 场景3：普通用户只能看到自己的客户
1. 使用普通用户账号登录（zhangsan@fengfancloud.com）
2. 进入"客户列表"，只能看到自己创建的客户
3. 进入"数据概览"，统计数据只包含自己的客户
4. 退出登录，使用管理员账号登录
5. 管理员可以看到所有客户，包括zhangsan创建的客户

### 场景4：非企业邮箱无法注册
1. 点击"立即注册"
2. 尝试使用非企业邮箱注册（例如：test@qq.com）
3. 系统提示"请使用公司企业邮箱（@fengfancloud.com）注册！"
4. 必须使用 xxx@fengfancloud.com 格式的邮箱才能注册成功

## 🔍 查重规则说明

### 普通用户添加客户时
- 检查**邮箱**是否已存在
- 检查**电话**是否已存在（如果填写了电话）
- 任一项重复都无法添加
- 编辑客户时，会排除当前客户自身进行查重

### 管理员添加客户时
- **无查重限制**
- 可以添加重复的邮箱或电话
- 适用于需要导入历史数据或特殊情况

## 💡 使用技巧

### 1. 多人协作
- 为客户分配多个销售负责人：王经理、李主管、张总监
- 使用逗号或顿号分隔

### 2. 产品标签
- 记录客户感兴趣的产品：企业管理软件，CRM系统，数据分析平台
- 使用逗号分隔

### 3. 数据统计
- 普通用户看到的是个人业绩
- 管理员看到的是团队整体业绩

### 4. 退出登录
- 点击右下角的退出图标
- 系统会清除登录状态

## ⚠️ 注意事项

1. **密码安全**
   - 本系统仅用于演示，密码以明文存储
   - 请勿使用真实重要密码

2. **数据存储**
   - 所有数据存储在浏览器LocalStorage中
   - 清除浏览器缓存会丢失所有数据
   - 建议定期导出备份（功能待开发）

3. **权限变更**
   - 新注册用户默认为普通角色
   - 如需升级为管理员，需要修改LocalStorage中的用户数据
   - 或联系系统管理员

4. **查重提示**
   - 如果提示客户已存在，请检查邮箱和电话
   - 管理员可以无视查重添加客户
   - 编辑客户时不会与自己冲突
